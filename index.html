Server administration

Depending on the technologies your website uses various issues can be anticipated. For node.js applications you'll want to restart the application when a fatal error occurs. This can be accomplished in various ways like using tools like systemd and upstart on Fedora resp. CentOS servers. Or using third-party tools created specifically for node.js applications. Another example is setting up SSL certificates to allow secure connections to your website.

Other aspects of server administration belong to my skillset as well. This includes setting up Apache, PHP, databases and the forementioned node.js.

As a part of my services I explain how to set up an account on your server that lets me connect in a secure manner. Among the instructions are rules to compartmentalise my access to your server.

Front-end development

Front-end development is what I'd call a buzzword in modern web development. Other terms include back-end or full stack development. What I would define as front-end development is using modern HTML and CSS in combination with client-side javascript to build a slick website. Several third-party libraries are often used such as Bootstrap for responsive design, Fontawesome for the great icons used on this site and fonts freely provided by google to get a pixel-perfect look and feel to your website.

On the javascript side one should look at libraries like jQuery for simple websites or more efficient frameworks for complicated applications. I thoroughly enjoy working with React since it almost forces you to write maintainable code.

I include comprehensible documentation that allows future developers you might work with can understand the flow in my code. This can save you money in the future since it should decrease the time needed to edit and/or expand your codebase.
